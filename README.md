# 🚀 Астра: Оффлайн-голосовой ассистент на Python

**Астра** — голосовой ассистент с локальным распознаванием речи (Vosk) и синтезом (pyttsx3). Он умеет управлять системой, искать информацию в интернете, воспроизводить музыку и фильмы, а также общаться с пользователем, обеспечивая конфиденциальность данных.
  

> 🔥 **Ключевые преимущества**:    
> - Использует локальную LLM-модель GigaChat (Сбер) через LLM Studio. Это обеспечивает полную конфиденциальность (данные не уходят в облако), а также дружелюбные и информативные ответы на русском языке с глубоким пониманием культуры и истории России  
> - Гибкая система команд

|-------------------------------|------------------------|-------------------|----------------|

| **Голосовое управление ПК/системой** |✅ выключение, горячие клавиши, текст голосом | ❌ ограничено | ❌ ограничено |
| **Мультимедиа (музыка/фильмы)** |✅ Hitmotop / Кинопоиск | ⚠️ Яндекс.Музыка (только по подписке, с авторизацией), кино не воспроизводит | ✅ Apple Music / Видео |

| **Управление браузером**       |✅ открытие, поиск, команды | ⚠️ ограничено | ⚠️ ограничено |

##  Возможности
- **Голосовое управление**: Просто скажите "Астра" и команду.  
- **Системные команды**:  
  - 🖥️ Выключение/перезагрузка ПК
  - ⏰ Узнать время и дату  
  - 🗑️ Очистка корзины  
  - ⌨️ Горячие клавиши
  - 📋 Ввод текста голосом 
- **Мультимедиа**:  
  - 🎵 Музыка и фильмы (Hitmotop и Кинопоиск, воспроизведение реализовано через Selenium)
- **Интернет**:  
  - 🌤️ Погода через wttr.in 
  - 🔍 Поиск в DuckDuckGo

## Требования    
- Windows 10/11
- Python 3.8+  
- Микрофон  
- Интернет *(только для некоторых функций: погода, новости, поиск и онлайн-медиа)*
- Google Chrome

## Структура проекта
```text
Voice_Assistant/
│
├── assistant_core.py             # Основной модуль ассистента

├── commands.json                 # Конфигурация команд ассистента

├── tts_module.py                 # Синтез речи

├── system_control.py             # Управление системой (выключение, горячие клавиши)

├── chat_gpt.py                   # Локальная интеграция ChatGPT
│
├── network/                      # Сетевая логика
│   ├── __init__.py               # Инициализация пакета
│   ├── network.py                # Основные сетевые операции
│   ├── music_service.py          # Воспроизведение музыки
│   └── film_service.py           # Воспроизведение фильмов
│
├── sounds/                       # Аудиофайлы для ассистента
│   ├── start.mp3
│   └── shutdown.mp3
├── main_applications/            # Ярлыки программ
│   ├── Google Chrome.lnk
│   ├── Telegram.lnk
│   └── WhatsApp.lnk
│
└── img/                          # Изображения проекта (баннер, иконки разных размеров и др.)
│
├── logs/                         # Логи работы ассистента
│   └── assistant.log
├── requirements.txt              # Зависимости проекта
```

## Настройка ИИ-ответов

Ассистент использует локальную LLM-модель с кастомизированным system_prompt<br>
**Правила:**
- **Краткие, но информативные** (1-2 предложения)  
- **Дружелюбные и естественные**
- **Предложение поиска в интернете, если нужны точные данные (курс валют и т.д.)**
- **Блокировку вредоносных, опасных или неэтичных запросов**  
- **Без сложных терминов**  
- **Структурированные** для списков/инструкций

## Установка
1. Клонирование репозитория:
 ```
https://github.com/fetgrigory/Voice_Assistant.git
   ```
2. Создайте виртуальное окружение для изоляции зависимостей проекта. 
   Используйте команду:
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - На Windows:
     ```bash
     venv\Scripts\activate
     ```
   - На macOS и Linux:
     ```bash
     source venv/bin/activate
     ```
4. Изоляция зависимостей:
   ```bash
     pip install -r requirements.txt
     ```
5. Запустите ассистента командой:
   ```bash
   python main.py
   ```
